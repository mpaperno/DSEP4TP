56 Current DSE plugin version number in integer format.
57
58 This is a "binary coded decimal" where<br />
59 `((MAJOR << 24) | (MINOR << 16) | (PATCH << 8) | BUILD)` (each part is limited to values 0-99).<br />
60 If formatted in hexadecimal this will match the version "number" that Touch Portal shows in the plugin's settings.<br />
61 E.g. `"1.2.30.4" == 16920580 == 0x01023004` or "01023004" as shown in Touch Portal. <br />
62 The possible "name" part of a version is ignored (the build number increments for each new pre-release alpha/beta version and again on the final release).
63 */
