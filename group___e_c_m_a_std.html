Standard JS Objects and Functions

This reference contains a list of standard ECMAScript objects, functions and properties supported by the plugin's JavaScript engine.

For full documentation see the MDN Global Objects reference.
For a detailed description see ECMA-262 specification. The scripting engine supports most features of ECMAScript level 7.
